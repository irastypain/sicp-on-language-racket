### Упражнение 1.21
С помощью процедуры `smallest-divisor` найдите наименьший делитель следующих
чисел: `199`, `1999`, `19999`.

#### Решение
Для `199` наименьший делитель - `199`, для `1999` - `1999`, а для `19999` - `7`.
